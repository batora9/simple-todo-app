# simple-todo-app

2024年度前期Web研究会で取り扱ったTODOアプリの改良版です。

## 使用技術

- サーバーサイド: Node.js(Hono.js)
- クライアントサイド: HTML, CSS, JavaScript

## 機能

- タスクの追加
- タスクの削除
- タスクの完了/未完了の切り替え
- タスク名の編集

## ローカルでの実行

1. このリポジトリをクローンします。

```bash
git clone
```

2. リポジトリの`server`ディレクトリに移動します。

```bash
cd simple-todo-app/server
```

3. 依存パッケージをインストールします。

```bash
npm install
```

4. サーバーを起動します。

```bash
node server.js
```

5. ブラウザで`/client/index.html`を開きます。
